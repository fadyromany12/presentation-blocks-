2. System Ecosystem Overview

KOMPASS is a unified ERP integrating 7 distinct operational pillars into a single source of truth:

1. Workforce Operations

A real-time Time & Attendance engine. Includes smart punch clock, Overtime requests, and adherence calculation logic.

2. Leave & Absence

End-to-end absence management. Features self-service requests, automated balance accruals, and "Sick Note" compliance.

3. HRIS, Assets & Lifecycle

The employee "Digital Twin." Manages profiles, Asset tracking, PII Change requests, and Offboarding workflows.

4. QA & Performance

A comprehensive talent management suite. Managers utilize dynamic coaching templates, track long-term scores, and reviews.

5. Recruitment (ATS)

Full-cycle Applicant Tracking. Manages Job Requisitions and candidates from application to one-click hiring.

6. Scheduling & Project Admin

Strategic planning tools. Supports bulk CSV roster imports, Project-specific break configs, and admin dashboards.

7. Finance & Administration

Enterprise governance. Features Financial Entitlements (Payroll), Dynamic RBAC (Role-Based Access), and Audit Trails.

Deep Dive: Workforce Operations

Agent Capabilities

  • Smart Punch Clock: Login, Break, and Lunch tracking with "Double-Punch" prevention.
  • Overtime Requests: New workflow for agents to request and track extra hours (Pre/Post shift).
  • Project Logging: Allocate time to specific client projects for billable hour tracking.
  • Real-Time Status: Visual timers showing elapsed time in current state.

System Logic & Automation

  • Adherence Engine: Auto-calculates Tardy and Early Leave based on Shift Start/End times.
  • Break Logic: Enforces project-specific break durations and windows (via Break_Config).
  • Auto-Sweeper: Detects "No Call, No Show" and auto-marks "Absent".
  • Aux Code Validation: Prevents unauthorized code usage (e.g., Coaching during Lunch).

Deep Dive: Leave & Absence

Employee Functions

  • Self-Service Portal: Submit Annual, Sick, or Casual leave requests.
  • Balance View: Real-time visibility of remaining entitlements.
  • Compliance Uploads: Mandatory PDF upload for "Sick Leave" justification.
  • Request History: Full audit trail of past approvals and rejections.

System Logic & Automation

  • Balance Validation: Instantly rejects requests exceeding available balance.
  • Approval Routing: Dynamic routing to Direct Manager or Project Manager based on hierarchy.
  • Auto-Deduction: Debits balances upon final approval.
  • Monthly Accrual: Automated script increments balances based on tenure.

Deep Dive: HRIS, Assets & Lifecycle

Profile & Lifecycle

  • Data Change Requests: Workflow for agents to request updates to PII (Address, IBAN).
  • Asset View: Agents can see hardware/assets assigned to them.
  • Resignation Request: Formal workflow for initiating self-resignation.
  • Hierarchy View: Interactive tree view of reporting lines.

System Logic & Automation

  • Offboarding Workflow: Manages termination logic, asset return status, and "Exit Date".
  • Data Guard: Locks sensitive fields (Salary, National ID) from direct editing.
  • Asset Inventory: Centralized tracking of all company assets linked to Employee IDs.
  • Reporting Lines: Prevents circular references in hierarchy changes.

Deep Dive: QA & Performance

Talent Management

  • Coaching Forms: Managers submit sessions using dynamic templates (e.g., "Q1 Sales").
  • Performance Reviews: Periodic (Q1/Q2/Annual) rating submission (1-5 scale).
  • Digital Acknowledge: Agents sign off on coaching feedback digitally.
  • Score History: Historical view of all QA scores and trends.

System Logic & Automation

  • Dynamic Templates: Renders forms based on active JSON templates (Category/Criteria).
  • Score Calculation: Auto-computes weighted averages for QA sessions.
  • Visibility Scoping: Ensures agents only see their own data; managers see their team's.
  • Review Cycle: Tracks completion status of quarterly reviews.

Deep Dive: Recruitment (ATS)

Hiring Tools

  • Requisitions: Managers can open new Job Requisitions (Positions).
  • Public Job Board: External portal for candidate applications.
  • Pipeline Manager: Kanban view for moving candidates (Applied -> Interview -> Offer).
  • Hiring Wizard: One-form hiring process to finalize contract details.

System Logic & Automation

  • One-Click Onboarding: "Hire" action creates User ID, Email, Drive Folders, and Balances instantly.
  • Duplicate Check: Identifies repeat applicants via Email/National ID.
  • Buffer Integration: Securely ingests external applications into the internal database.

Deep Dive: Scheduling & Project Admin

Admin Tools

  • Project Dashboard: View headcount and stats per project/campaign.
  • Bulk Importer: Upload hundreds of shifts via CSV with validation.
  • Manual Schedule: Fine-tune individual rosters and days off.
  • Project Creation: Create new Projects and assign Managers.

System Logic & Automation

  • Gap Analysis: Distinguishes "Scheduled Day Off" vs. "Absent" automatically.
  • Project Configs: Applies project-specific settings (e.g., distinct break times per campaign).
  • Conflict Check: Prioritizes approved Leave over new schedule uploads.

Deep Dive: Finance & Administration

Superadmin Controls

  • Financial Entitlements: Manage payroll additions (Bonuses, Allowances, Variable Pay).
  • User Gateway: Approve/Deny new user registrations.
  • Role Management: Approve requests for role upgrades (Agent -> Admin).
  • Announcements: Broadcast system-wide messages to all users.

System Logic & Automation

  • Dynamic RBAC: Permission matrix (RBAC_Config) controls access to every module.
  • Audit Trails: Logs every critical action (Entitlement updates, Punches, Edits).
  • Session Security: Validates user role and status on every API call.

  • Disaster Recovery: Implementation of automated backup and snapshot strategies.

5. Resource Forecast

Projected capacity requirements for server sizing based on 200 active users.

Structured Data (Relational)

  • Daily Punches: ~1,600 rows/day
  • Annual Growth: ~600k rows/year
  • Storage Requirement: < 1 GB / year

Low storage footprint; high transaction speed required.

Unstructured Data (Object Storage)

  • Documents (PDF/IMG): ~200 MB / week
  • Annual Growth: ~10 - 50 GB
  • Storage Requirement: High Capacity

Strategic Recommendation: Utilize Google Cloud Storage buckets for files to keep the Database performant.
