KOMPASS Migration

From Apps Script Prototype to Cloud Enterprise Application

Goal: Migrate our 7-module HR Super-App to a scalable Node.js & PostgreSQL architecture on Google Cloud.

Slide 1/9
KOMPASS

1. Workforce Management (WFM)

Replacing the "Adherence" and "Punch" sheets with a high-speed SQL database.

  • Smart Punch Clock: Validates "Login", "Break", and "Logout" in real-time. Prevents double-punching and sequential errors.
  • Adherence Engine: Automatically compares Actual Punch Time vs. Scheduled Shift to calculate Tardy and Early Leave seconds.
  • Project Tracking: Users select active projects (e.g., "Lenovo Support") to log billable hours per session.
  • Auto-Sweeper: A background job that runs daily to mark employees as "Absent" if they were scheduled but didn't punch in.

2. Leave & Absence

Moving from "Leave Requests" sheet to a relational transaction model.

  • Balance Validation: Instant check against Annual/Sick/Casual balances before submission.
  • Dynamic Routing: Workflow routes requests: Agent → Project Manager → Direct Manager → Approved.
  • Sick Note Compliance: Enforces PDF uploads for sick leave. Files will move from Drive Folders to Google Cloud Storage.
  • Monthly Accrual: Automated script increments leave balances based on employee tenure (1+ year vs 10+ years).

3. HRIS & Digital Files

Centralizing Employee Core Data and PII.

  • Digital Employee Files: We currently auto-generate Drive folders for every user (Payslips, Contracts).
  • Self-Service Profile: Agents update their own Address/Phone. Sensitive changes (IBAN) trigger an approval request log.
  • Hierarchy Visualization: A recursive "Tree View" allowing managers to see their direct and indirect reports.
  • Movement Requests: Formalized workflow for changing an employee's supervisor (Reporting Line changes).

4. Talent & Quality (QA)

Digitizing the Coaching and Review process.

  • Dynamic Templates: Managers select templates (e.g., "Q1 Sales") which populate specific scoring criteria.
  • Performance Reviews: Quarterly/Annual review forms with 1-5 ratings stored historically.
  • Feedback Loop: Agents must digitally "Acknowledge" coaching sessions, creating a timestamped audit trail.

5. Recruitment (ATS)

Managing the "Candidate to Employee" lifecycle.

  • Public Job Board: External facing application form for candidates.
  • Pipeline Management: Admin view to move candidates through stages (New → Interview → Offer).
  • One-Click Hiring: The "Hire" button automatically generates the Employee ID, creates the DB record, and initializes Drive folders.

6. Scheduling & Security

Enterprise-grade control and planning.

  • Shift Planning: Admin interface to assign shifts and "Day Off" status.
  • Bulk CSV Import: Ability to upload hundreds of schedules at once.
  • RBAC Security: Strict Role-Based Access Control. Agents see self; Managers see team; Superadmins see all.
