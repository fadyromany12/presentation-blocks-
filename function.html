2. System Ecosystem Overview

KOMPASS is a unified ERP integrating 7 distinct operational pillars into a single source of truth:

1. Workforce Operations

A real-time Time & Attendance engine. It includes a smart punch clock, adherence calculation logic (Tardy/Overtime), and live floor monitoring dashboards.

2. Leave & Absence

End-to-end absence management. Features self-service requests, automated balance accruals, multi-tier approval workflows, and "Sick Note" compliance.

3. HRIS & Digital Files

The employee "Digital Twin." Automatically generates Drive folders for every user, manages reporting lines hierarchy, and secures PII data.

4. QA & Performance

A comprehensive talent management suite. Managers utilize dynamic coaching templates, track long-term scores, and enforce digital acknowledgement loops.

5. Recruitment (ATS)

Full-cycle Applicant Tracking System. Manages candidates from public application to interview stages, with one-click hiring that provisions all assets instantly.

6. Scheduling & Planning

Strategic workforce planning tools. Supports bulk CSV roster imports, shift pattern enforcement, and intelligent logic distinguishing "Day Off" from "Absent".

7. Administration & Security

Enterprise-grade governance foundation. Features Role-Based Access Control (RBAC), comprehensive audit trails for compliance, and system-wide broadcasting tools.

Deep Dive: Workforce Operations

Agent Capabilities

  • Digital Punch Clock: One-click interface for Login, Breaks, Lunch, and Logout.
  • Project Allocation: Select specific client projects (e.g., "Lenovo Support") to log billable hours.
  • Real-Time Status: Visual feedback on current state (e.g., "On Lunch - 25 mins elapsed").
  • Aux Code Tracking: Log non-production time like "Coaching" or "Meeting" accurately.

System Logic & Automation

  • Adherence Engine: Calculates deviation (Tardy/Early Leave) by comparing actual punches vs. scheduled shifts instantly.
  • Validation Logic: Prevents sequential errors (e.g., trying to "Login" when already logged in).
  • Auto-Sweeper: Background job that identifies scheduled employees who failed to punch and marks them "Absent".
  • Live Dashboard: Pushes real-time state changes to the Manager's Command Center.

Deep Dive: Leave & Absence

Employee Functions

  • Self-Service Portal: Submit Annual, Sick, or Casual leave requests directly.
  • Balance Visualization: View real-time remaining days for each leave type before applying.
  • Compliance Uploads: Mandatory PDF upload interface for "Sick Leave" justification.
  • Request History: Track status of all past and pending applications.

System Logic & Automation

  • Pre-Submission Validation: Instantly rejects requests if the user has insufficient balance.
  • Smart Routing: Dynamically routes approval to Project Manager or Direct Manager based on hierarchy.
  • Auto-Deduction: Debits leave balances immediately upon final approval.
  • Schedule Integration: Automatically updates the shift roster to reflect "On Leave" status for approved dates.

Deep Dive: HRIS & Digital Files

Profile Management

  • Digital Profile: Employees can view and update contact details (Address, Phone, Emergency Contact).
  • Hierarchy View: Interactive "Tree View" allowing managers to navigate their direct and indirect reports.
  • Movement Requests: Formal workflow for requesting supervisor changes (Reporting Line updates).
  • Document Access: Secure links to personal Payslips and Contracts.

System Logic & Automation

  • Asset Provisioning: Automatically generates and names Google Drive folders for new hires.
  • Sensitive Data Guard: Monitors critical fields (IBAN, Name); changes trigger an audit log/approval request.
  • Structure Integrity: Prevents circular reporting lines when processing hierarchy moves.
  • Offboarding Protocol: Standardized termination sequence to revoke access and archive data.

Deep Dive: QA & Performance

Manager & Agent Tools

  • Template Selection: Managers choose from dynamic templates (e.g., "Q1 Sales", "Soft Skills").
  • Scorecard Submission: Interactive forms for rating criteria and adding qualitative feedback.
  • Digital Acknowledgement: Agents must sign off on feedback digitally, ensuring alignment.
  • History Review: Access to a chronological log of all performance scores and reviews.

System Logic & Automation

  • Dynamic Rendering: Generates form fields on the fly based on the selected JSON template.
  • Score Calculation: Auto-computes weighted averages and percentages for QA sessions.
  • Feedback Loop Enforcement: Tracks "Pending Acknowledgement" status to ensure compliance.
  • Data Isolation: Ensures agents only see their own reviews, while managers see their team's.

Deep Dive: Recruitment (ATS)

Recruitment Functions

  • Public Job Board: External-facing application portal for candidates to submit data and CVs.
  • Pipeline View: Kanban-style status management (Applied → Interview → Offer → Hired).
  • Candidate Search: Filterable database of all applicants and their history.
  • Hiring Wizard: A simplified form to finalize contract details for selected candidates.

System Logic & Automation

  • One-Click Hiring: Triggering "Hire" creates the DB record, generates Employee ID, and initializes balances instantly.
  • Duplicate Detection: Flags candidates who have applied previously to prevent redundancy.
  • External Integration: Fetches applications from a secure buffer sheet to protect internal data.
  • Status Transitioning: Automates the lifecycle movement from "Candidate" to "Employee".

Deep Dive: Scheduling & Planning

Planning Tools

  • Bulk CSV Import: Upload rosters for hundreds of agents at once using a standardized template.
  • Manual Editor: Fine-tune individual shifts or override statuses for specific days.
  • Status Designation: Explicitly mark days as "Shift", "Day Off", or "Leave".
  • Template Download: Download current roster formats for offline editing.

System Logic & Automation

  • Gap Detection: Distinguishes between a "Scheduled Day Off" (No punch needed) and "Absent" (Missed shift).
  • Format Validation: Parses incoming CSVs for date/time errors before committing to the database.
  • Conflict Resolution: Prioritizes "Approved Leave" over uploaded shift data to prevent errors.

Deep Dive: Administration & Security

Superadmin Controls

  • User Registration Gateway: Review and approve/deny new account requests before activation.
  • Role Management: Process requests for role elevation (e.g., Agent to Admin).
  • Announcements: Create and broadcast system-wide banners to all active users.
  • Template Builder: Create and modify coaching forms without code changes.

System Logic & Automation

  • RBAC Enforcement: Strict Role-Based Access Control ensuring secure data partitioning.
  • Audit Logging: Immutable logs of every critical action (Timestamp, User ID, Action Type).
  • Session Security: Validates user identity against the database on every API call.

3. Migration Responsibilities

  • Database Administration: Installation and initial setup of the PostgreSQL instance.
  • Network Security: Management of firewalls, SSL certificates, and secure access protocols.
  • Disaster Recovery: Implementation of automated backup and snapshot strategies.

5. Resource Forecast

Projected capacity requirements for server sizing based on 200 active users.

Structured Data (Relational)

  • Daily Punches: ~1,600 rows/day
  • Annual Growth: ~600k rows/year
  • Storage Requirement: < 1 GB / year

Low storage footprint; high transaction speed required.

Unstructured Data (Object Storage)

  • Documents (PDF/IMG): ~200 MB / week
  • Annual Growth: ~10 - 50 GB
  • Storage Requirement: High Capacity

Strategic Recommendation: Utilize Google Cloud Storage buckets for files to keep the Database performant.
